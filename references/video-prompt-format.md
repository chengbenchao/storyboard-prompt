# Seedance 2.0 视频分镜提示词格式规范

## 标准结构（单镜头）
```
[时长]s | [镜头类型/运镜] | [画面核心内容] | [光影风格] | [画质要求] | [约束条件]
```

## 字段释义（必填项）
| 字段 | 说明 | 示例 |
|------|------|------|
| 时长 | 单镜头时长，单位秒，整数，符合分镜表单镜≤5s要求 | 3 |
| 镜头类型/运镜 | 明确运镜方式、景别、视角 | 特写推镜 / 中景固定 / 全景横移 / 仰拍拉远 |
| 画面核心内容 | 精准描述画面内容：人物动作、位置、道具、环境、交互关系 | 古装女帝站在金銮殿龙椅旁，手指下方跪地大臣，眼神凌厉 |
| 光影风格 | 明确整体光影色调、氛围感 | 冷光宫廷风，暗调背光，金色龙纹反光 |
| 画质要求 | Seedance 2.0专用画质参数 | 4K 120fps 电影级质感 无畸变 无AI artifacts |
| 约束条件 | 明确禁止的内容 | 无水印 无多余人物 无穿帮元素 |

## 适配不同场景的扩展规则
### 1. 对话场景
```
4s | 中景正反打 | 男主坐在餐桌旁抬头看向对面女主，嘴角上扬微笑 | 暖黄室内打光，日系治愈风 | 4K 60fps 写实质感 | 动作自然流畅 无夸张表情
```
### 2. 动作场景
```
3s | 低角度跟拍 | 黑衣反派持刀冲向男主，男主侧身闪避，刀身擦过男主衣角 | 冷雨夜暗调，街道路灯侧光 | 4K 120fps 动态模糊自然 | 动作连贯 无卡顿
```
### 3. 转场场景
```
2s | 快速闪切 | 手中酒杯落地碎裂的特写 → 溅起的红酒滴在空中 | 暖红光影，慢动作效果 | 4K 60fps 细节清晰 | 转场自然 无突兀感
```

## Seedance 2.0 官方推荐参数组合
| 视频类型 | 推荐参数 |
|----------|----------|
| 短剧正片 | 4K 60fps 电影级质感 写实风格 无畸变 |
| 预告片 | 4K 120fps 高对比度 动态模糊自然 |
| 特效镜头 | 4K 60fps 细节拉满 无AI artifacts |

## 注意事项
1. 单条提示词总长度控制在200字以内，避免冗余
2. 优先使用具象描述，避免抽象形容词（如"好看"→"皮肤白皙通透，妆容精致"）
3. 动作描述要精准到具体肢体动作，方便AI生成流畅动态
4. 所有提示词统一使用中文，符合Seedance 2.0中文模型优化要求
