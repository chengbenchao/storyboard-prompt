# Seedance 2.0 视频分镜提示词格式规范
> 适配字节跳动Doubao Seedance 2.0官方2026.04最新版提示词指南，专门优化短剧视频生成效果

## 官方标准结构（单镜头/必填）
```
[时长]s | [镜头类型/运镜] | [画面核心内容] | [动作逻辑] | [光影风格] | [画质要求] | [约束条件] | [可选：卡点标记]
```

## 字段释义（严格遵循官方要求）
| 字段 | 官方要求 | 示例 |
|------|----------|------|
| 时长 | 单镜头时长，单位秒，整数，短剧场景要求单镜≤5s，总单集时长60s±5s | 3 |
| 镜头类型/运镜 | 必须明确景别+运镜方式+视角，禁止模糊描述 | 脸部特写慢推 / 中景固定 / 全景低角度横移 / 仰拍快速拉远 |
| 画面核心内容 | 必须具象描述：人物穿着/表情/位置、道具细节、环境、人物交互关系，禁止抽象形容词 | 穿明黄色龙袍的女帝站在金銮殿龙椅旁，右手食指指向下方跪地的大臣，眼神凌厉充满怒意 |
| 动作逻辑 | ✅ 官方新增必填项：明确所有物体/人物的运动轨迹、速度、方向，确保动作连贯自然 | 女帝手指缓慢向下移动，大臣身体微微发抖，头埋得更低 |
| 光影风格 | 明确光源类型、色调、氛围感、阴影效果 | 冷光宫廷风，顶部主光源打在女帝身上，暗调背光，龙袍上的金色龙纹有反光效果 |
| 画质要求 | 必须使用Seedance 2.0官方支持的参数，禁止自定义参数 | 4K 60fps 电影级质感 写实风格 无畸变 无AI artifacts |
| 约束条件 | 明确禁止出现的内容，必须包含官方要求的默认约束 | 无水印 无多余人物 无穿帮元素 动作不僵硬 面部无扭曲 |
| 卡点标记 | 可选，短剧场景专用：标记该镜头对应的卡点位置 | 第15s爽点 / 第55s反转钩子 |

## 官方明确禁止的描述/避坑指南
1. ❌ 禁止使用模糊形容词：好看、漂亮、高质量、高清等，必须替换为具象描述
   ✅ 正确示例：皮肤白皙通透，妆容精致，服装纹理清晰
2. ❌ 禁止出现英文描述：Seedance 2.0中文模型对英文支持差，所有描述必须使用中文
3. ❌ 禁止长度超过200字：单条提示词控制在150-200字以内，避免冗余
4. ❌ 禁止动作描述模糊：必须明确到具体肢体动作，比如"男主很生气"→"男主拳头紧握，眉头紧皱，牙齿咬着下唇"
5. ❌ 禁止使用不支持的画质参数：官方仅支持2.7K/4K分辨率，30/60/120fps帧率，其他参数会被忽略
6. ✅ 官方推荐：每个提示词至少包含3个以上的细节描述，生成效果提升40%以上
7. ✅ 官方推荐：短剧场景优先使用竖屏9:16比例，符合短视频平台播放要求
