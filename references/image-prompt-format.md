# AI生图提示词规范

## 核心：夏导引擎10维度结构

复用 `xiameng-prompt-engineer`（夏导引擎v2.2），10维度：

| 维度 | 说明 |
|------|------|
| ①景别 | 大远景/远景/全景/中景/近景/特写/大特写 |
| ②拍摄高度 | 平拍/仰拍/俯拍/鸟瞰/低机位/高机位 |
| ③拍摄方向 | 正面/侧面/斜侧/背面/过肩 |
| ④焦段 | **严格按景别匹配**（见下表） |
| ⑤光位 | 顺光/侧光/侧逆光/逆光/顶光/伦勃朗光 |
| ⑥光比 | 1:1~2:1柔和 / 3:1~4:1适中 / 6:1~8:1戏剧 |
| ⑦主体 | 姿态+表情+动作+位置+**细节质感** |
| ⑧环境 | 背景+道具+材质+空间关系+**景深控制+前景** |
| ⑨色温 | 环境色温K + 面部色温K（严格按铁律） |
| ⑩氛围 | 情绪关键词+**画质词后缀** |

## 场景/道具/分镜关键帧生图格式

按场景类型选择对应色温，按景别选择对应焦段，主体描述中必须包含：
- 环境/场景
- 人物位置/动作
- 表情/情绪
- 光影/色彩
- 关键道具

景深控制：前景虚化/中景深/深景深（按景别选择）
