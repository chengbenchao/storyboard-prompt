# 表格输出规范

## 一、分镜表

### 字段定义

| 字段 | Bitable类型 | 必填 | 说明 |
|------|------------|------|------|
| 镜头 | Number | ✅ | 镜号 1,2,3... |
| 景别 | SingleSelect | ✅ | 大远景/远景/全景/中景/近景/特写 |
| 运镜 | SingleSelect | ✅ | 固定/推/拉/摇/跟拍/俯拍/仰拍/闪切/慢放 |
| 人物 | Text | ✅ | 镜头内角色 |
| 台词 | Text | ⭕ | 对白/旁白/OS/OV |
| 画面 | Text | ✅ | 详细视觉描述 |
| 时长 | Text | ✅ | ≤5秒，如 3秒 / 1.5秒 |

### 操作流程

1. `feishu_bitable_create_app` 建表
2. `feishu_bitable_create_field` 建字段
3. `feishu_bitable_create_record` 逐镜写入
4. `lark-cli docs +update` 同步到Docx

### 规则

- 每集 10-13 镜
- 每镜 ≤5 秒
- 避免连续 3 镜同景别

## 二、场景表

### 字段定义

| 字段 | Bitable类型 | 必填 | 说明 |
|------|------------|------|------|
| 场景名 | Text | ✅ | 如"孤城城头·黄沙漫天·黄昏" |
| 集数 | Text | ✅ | 出场集数，如"第1、2、8集" |
| 地点 | Text | ✅ | 如"孤城城头" |
| 时间 | SingleSelect | ✅ | 日/夜/黄昏/黎明 |
| 内外景 | SingleSelect | ✅ | 内景/外景 |
| 光影氛围 | Text | ✅ | 如"侧逆光+暖金轮廓光+沙尘粒子" |
| 出场人物 | Text | ⭕ | 该场景出场角色列表 |
| 备注 | Text | ⭕ | 补充说明 |

### 操作流程

1. 从剧本提取所有场景
2. `feishu_bitable_create_app` 建表
3. `feishu_bitable_create_field` 建字段
4. `feishu_bitable_create_record` 逐场景写入

---

## 三、角色人设表

### 字段定义

| 字段 | Bitable类型 | 必填 | 说明 |
|------|------------|------|------|
| 角色名 | Text | ✅ | 如"顾长安" |
| 角色标签 | Text | ✅ | 如"R1"，用于视频提示词引用 |
| 年龄 | Number | ✅ | |
| 身份 | Text | ✅ | 如"安西孤城守将" |
| 外貌特征 | Text | ✅ | 面部/发型/身材 |
| 服装 | Text | ✅ | 各阶段服装描述 |
| 性格 | Text | ⭕ | 性格关键词 |
| 出场集数 | Text | ✅ | |
| 备注 | Text | ⭕ | |

### 生成流程

1. 建 Bitable 表（人设表结构）
2. 为每个角色生成 4 条生图提示词（见 `image-prompt-format.md`）：
   - 正视图（全身，50mm，纯白背景）
   - 侧视图（全身，50mm，纯白背景）
   - 背视图（全身，50mm，纯白背景）
   - 特写（面部，85mm，纯白背景）
3. 写入 Docx
